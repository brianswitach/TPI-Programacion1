# Gestión de Datos de Países — TPI Programación 1

Aplicación de consola en Python para gestionar información de países: agregar, actualizar, buscar, filtrar, ordenar y calcular estadísticas.

## Requisitos
- Python 3.8+

## Instalación
1. Clonar o descargar la carpeta del proyecto.
2. Asegurarse de que `dataset.csv` esté en el mismo directorio que `tpi.py`.

## Uso
```bash
python tpi.py
```
El menú guía las operaciones. Se puede cargar el CSV inicial, operar sobre los datos en memoria y guardar cambios.

## Ejemplos de entradas/salidas
- Búsqueda por nombre: ingresar parte del nombre devuelve coincidencias.
- Filtros: por continente, por rango de población o de superficie.
- Ordenamientos: por nombre/población/superficie asc/desc.
- Estadísticas: mayor/menor población, promedios, cantidad de países por continente.
